## 4주차 - 상상의 현실화 – 그들이 만들어 가는 세상

### MIT Sixth Sense 연구팀이 만들고자 했던 것은?
- Why?\
실생활과 디지털 세계를 연결하여 생활을 편하게 만들어 보고자 하였다.
- What?\
현실 물체를 통해 디지털 정보를 얻을 수 있는 체계를 만들었다. 그렇게 디지털 세계와 현실 세계의 경계를 흐리게 하였다.
- How?\
비행기 표를 올려두면 항공편의 정보가 나오거나 펜을 움직여 3차원 그림을 그리는 등의 방법으로 구현하였다.

### As-Is vs To-Be는 어떻게?
#### 기존 컴퓨터의 입력, 출력 장치를 어떠한 방식으로 변경하였나요??
- 키보드는?\
포스트잇이나 종이 등에 직접 쓰는 방식으로 대체하였다.
- 마우스는?\
센서를 통해 손으로 펜이나 손 자체를 움직이는 방식으로 대체하였다.
- 모니터는?\
빔 프로젝터로 몸이나 종이 등에 비추는 방식으로 대체하였다.

### MIT Sixth Sense Project 팀이 만든 내용이 현재에는 어떠한 장비로, 어떻게 구현이되었을까?
- 현재의 기술과 약 15년 전에 MIT에서 진행한 기술의 발전 모습은?\
비록 아직은 실제로 현실 세계와 디지털 세계를 완전히 이은 듯한 모습은 아니지만 스마트폰을 통해 터치만 하면 거의 모든 것을 할 수 있게끔 구현되었다.
